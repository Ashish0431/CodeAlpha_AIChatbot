CodeAlpha FAQ Chatbot 🤖

Overview

This project is a simple AI-based FAQ Chatbot developed as part of the CodeAlpha Artificial Intelligence Internship.

The chatbot answers user questions by matching them with predefined FAQs using text similarity techniques. It provides quick responses through a user-friendly chat interface.

Features

- Interactive chat interface
- FAQ-based question answering
- Text preprocessing and cleaning
- Similarity matching for user queries
- Responsive design
- Works directly in the browser

Technologies Used

- HTML5
- CSS3
- JavaScript

Project Structure

CodeAlpha_FAQChatbot/

├── index.html

├── style.css

├── script.js

└── faqs.js

How It Works

1. User enters a question.
2. The chatbot preprocesses the text.
3. The system compares the question with stored FAQs.
4. The most relevant answer is returned.
5. The response is displayed in the chat window.

Future Improvements
